# MCP Admin UI 启动说明

## 前置要求

- Node.js >= 18.0.0
- npm >= 9.0.0

## 安装步骤

### 1. 进入项目目录

```bash
cd /Users/hamm/Desktop/mcp-platform/mcp-admin-ui
```

### 2. 安装依赖

```bash
npm install
```

或者使用其他包管理器：

```bash
# 使用 yarn
yarn install

# 使用 pnpm
pnpm install
```

### 3. 启动开发服务器

```bash
npm run dev
```

启动成功后，访问 http://localhost:3000

### 4. 构建生产版本

```bash
npm run build
```

构建产物将输出到 `dist` 目录。

### 5. 预览生产构建

```bash
npm run preview
```

## 默认登录信息

由于是演示项目，您可以使用任意用户名和密码登录，或者根据实际后端 API 配置登录接口。

## 功能模块

1. **仪表盘** (`/dashboard`) - 查看系统统计数据和活动日志
2. **服务管理** (`/services`) - 管理 MCP 服务
3. **API Key 管理** (`/apikeys`) - 管理 API 密钥
4. **实例管理** (`/instances`) - 管理服务实例

## 后端 API 配置

项目默认配置后端 API 地址为 `http://localhost:8080`，通过 Vite 代理转发。

如需修改，请编辑 `vite.config.ts` 文件中的 `server.proxy` 配置。

## 常见问题

### 端口被占用

如果 3000 端口被占用，可以在 `vite.config.ts` 中修改端口号，或者使用：

```bash
npm run dev -- --port 3001
```

### 依赖安装失败

尝试清除缓存后重新安装：

```bash
rm -rf node_modules package-lock.json
npm install
```

### TypeScript 报错

确保安装了正确的 TypeScript 版本：

```bash
npm install -D typescript@^5.4.0
```
