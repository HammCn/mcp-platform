## 修复的问题

### 1. Maven 编译参数问题
**问题**: Spring Data Pageable 参数名无法识别

**原因**: Maven 编译时未添加 `-parameters` 参数

**解决方案**: 在 `mcp-admin/pom.xml` 中添加 Maven compiler 插件配置
```xml
<plugin>
    <groupId>org.apache.maven.plugins</groupId>
    <artifactId>maven-compiler-plugin</artifactId>
    <configuration>
        <parameters>true</parameters>
    </configuration>
</plugin>
```

### 2. CSRF 保护问题
**问题**: POST 请求返回 401 未授权

**原因**: Spring Security 默认启用 CSRF 保护

**解决方案**: 创建 `SecurityConfig` 配置类，对 API 路径禁用 CSRF
```java
@Configuration
@EnableWebSecurity
public class SecurityConfig {
    @Bean
    public SecurityFilterChain filterChain(HttpSecurity http) throws Exception {
        http
            .csrf(csrf -> csrf.ignoringRequestMatchers("/api/**"))
            .authorizeHttpRequests(auth -> auth
                .requestMatchers("/api/**").authenticated()
                .anyRequest().authenticated()
            )
            .httpBasic(basic -> {})
            .sessionManagement(session -> session.disable());
        return http.build();
    }
}
```
