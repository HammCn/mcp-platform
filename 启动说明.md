## 四、启动应用

### 方式 1：使用启动脚本
```bash
cd /Users/hamm/Desktop/mcp-platform
chmod +x start.sh
./start.sh
```

### 方式 2：手动启动
```bash
cd /Users/hamm/Desktop/mcp-platform

# 编译项目
mvn clean install -DskipTests

# 启动应用
mvn spring-boot:run -pl mcp-admin
```

### 方式 3：使用 JAR 包启动
```bash
cd /Users/hamm/Desktop/mcp-platform

# 打包
mvn clean package -DskipTests

# 启动
java -jar mcp-admin/target/mcp-admin-1.0.0-SNAPSHOT.jar
```

## 七、常见问题

### 7.1 端口被占用
错误：`Port 8080 is already in use`

解决：修改 `application.yml` 中的端口
```yaml
server:
  port: 8081  # 改为其他端口
```

### 7.2 数据库连接失败
错误：`Communications link failure`

解决：
1. 检查 MySQL 是否启动
2. 确认数据库 `mcp_platform` 已创建
3. 确认用户名密码正确

### 7.3 Redis 连接失败
错误：`Unable to connect to Redis`

解决：
1. 启动 Redis：`redis-server`
2. 检查 Redis 配置

### 7.4 编译失败
错误：`Could not resolve dependencies`

解决：
1. 检查网络连接
2. 清理 Maven 缓存：`mvn clean`
3. 使用国内镜像（见下方配置）

## 八、Maven 国内镜像配置

如果下载依赖慢，可以在 `pom.xml` 中添加阿里云镜像：

```xml
<repositories>
    <repository>
        <id>aliyun</id>
        <name>Aliyun Maven</name>
        <url>https://maven.aliyun.com/repository/public</url>
    </repository>
</repositories>

<pluginRepositories>
    <pluginRepository>
        <id>aliyun</id>
        <name>Aliyun Maven</name>
        <url>https://maven.aliyun.com/repository/public</url>
    </pluginRepository>
</pluginRepositories>
```
